Why we built this

Created by Ashish Kumar

The GST Price Calculator was designed to help businesses and individuals quickly extract actual base prices from total amounts that already include Goods and Services Tax.

Instead of doing complex manual math calculations to reverse-engineer the 18% GST addition, this tool handles it automatically with perfect precision—allowing you to deduct discounts securely before recalculating the final price point.

Tech Stack
Frontend HTML5
Styling Vanilla CSS3
Logic JavaScript (ES6)

GST rate used: 18%  |  All values rounded to 2 decimal places
